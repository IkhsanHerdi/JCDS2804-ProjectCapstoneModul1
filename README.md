Project Capstone Modul 1 - Library Application

**Pengguna Aplikasi :
1. Admin
2. User**

**Fitur Pengguna Aplikasi**
* **Fitur Admin**
  * Lihat Katalog Buku
  * Pengembalian Buku
  * Lihat buku yang Dipinjam
  * Tambah Buku Baru
  * Update Informasi Buku
  * Hapus Buku
  * Cari Buku
  * Recycle Bin
  * Buku yang paling sering Di sewa
  * History Peminjaman Buku
  * Logout

* **Fitur User**
  * Lihat Katalog Buku
  * Sewa Buku
  * Lihat buku yang Dipinjam
  * Logout
